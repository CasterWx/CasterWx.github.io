你好，世界——欢迎来到新站点.

2026年5月1日阅读 2 分钟hello-wo
站务引言

为什么要重建？

本站在 2020 年之后就没有更新过了——当时只是一个手写的静态 HTML 页面。现在用 Next.js、TypeScript 和 Tailwind CSS 全面重建，原因有三：

  • 内容驱动：所有文章和项目页面都用 MDX 编写——Markdown 内嵌 React 组件。编辑体验与写文本文件一样简单。
  • 静态导出：整个站点编译为纯 HTML/CSS/JS 文件，部署到 GitHub Pages。无服务器、无数据库、无运行时依赖。
  • 类型安全：TypeScript 在构建时捕获错误，而非线上才发现问题。

会写些什么？

软件工程实践、工具链心得、建造过程中学到的经验教训。站点同时包含项目展示和关于页面。
